	TIPOS DE SCOPE (Tipos de alcance)
	SCOPE global
	Se puede llamar a la variable en toda la extensión del programa.

	SCOPE local
	Las variables soloson llamadas en la función en particular en la que son declaradas y usadas.

	VARIABLE Hosting (Var y functions only*)
	Escan de todas las variables y asignar valores.

VARIABLES
var - Usado para declarar una variable global o local. Se puede inicializar con un valor.
let - Usado para incializar un block scope, local variable. Puede ser incializada con un valor.
const - Para declarar una block scoped o read only constant. Debe tener un valor previo.
